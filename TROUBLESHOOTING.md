### üìÑ 1. Erreurs de Parsing CSV

#### **Probl√®me :** "Cannot convert to numeric" ou colonnes mal s√©par√©es

**üîç Cause :** D√©limiteur CSV incorrect (`;` vs `,`)

**‚öôÔ∏è Solution Automatique :**

```python
# D√©tection automatique impl√©ment√©e dans les dashboards
def detect_delimiter(file_path):
    with open(file_path, 'r') as f:
        first_line = f.readline()
        if ';' in first_line and first_line.count(';') > first_line.count(','):
            return ';'
        return ','
```

**üõ†Ô∏è Solution Manuelle :**

```python

    df = df_semicolon
else:
    df = df_comma
```
